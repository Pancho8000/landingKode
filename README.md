## landingKode

Landing page de marca personal para Ingeniero Informático, centrada en mostrar habilidades, proyectos reales y servicios de desarrollo web end‑to‑end, con soporte multilenguaje (ES/EN).

## Tecnologías

- HTML5 y CSS3
- JavaScript
- PHP (entorno XAMPP)
- Bootstrap (en proyectos referenciados)

## Características

- Sección de héroe con etiquetas de rol (Ingeniero Informático, Full‑Stack, End‑to‑End)
- Sección Sobre mí con foco en soluciones y contexto HSE, inventario y portales
- Listado de habilidades técnicas
- Sección de servicios orientados a equipos y proyectos digitales
- Portafolio de proyectos (notiJS, sistemaInventario, landingKode, plataforma HSE Puerto Punta Chungo)
- Cambio de idioma ES/EN mediante diccionario de traducciones en JavaScript
- Enlaces a LinkedIn, GitHub y contacto por correo

## Ejecución en local (XAMPP)

1. Clonar el repositorio en la carpeta `htdocs`:
   ```bash
   cd C:\xampp\htdocs
   git clone https://github.com/Pancho8000/landingKode.git
   ```
2. Asegurar que Apache esté iniciado en XAMPP.
3. Acceder desde el navegador a:
   - `http://localhost/landingKode/public/index.html`
   - o `http://localhost/landingKode/` si el servidor respeta el `DirectoryIndex`.

## Estructura básica

- `public/` contiene el `index.html` de la landing.
- `src/assets/styles/` contiene los estilos principales (`main.css`).
- `src/assets/scripts/` contiene la lógica de idioma y enlaces (`main.js`).
